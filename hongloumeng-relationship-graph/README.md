# 红楼梦人物关系图生成专家

> WorkBuddy 技能 · 生成《红楼梦》四大家族人物关系图

## 功能特性

- 📚 **知识库完备**：覆盖贾、史、王、薛四大家族及金陵十二钗
- 🔗 **关系可视化**：基于 ECharts 生成交互式关系图
- 📊 **结构化数据**：输出 JSON 格式，便于二次开发
- 🎨 **古风配色**：专属设计美学，还原红楼梦意境

## 文件结构

```
hongloumeng-graph/
├── SKILL.md                    # 技能主文件
├── README.md                   # 说明文档
├── assets/
│   └── graph-template.html     # ECharts 关系图模板
└── references/
    └── characters.md           # 详细人物关系参考
```

## 使用方法

### 在 WorkBuddy 中触发

询问以下问题即可自动触发：

- "红楼梦人物关系"
- "生成关系图"
- "宝黛钗关系"
- "贾府人物"
- "金陵十二钗"

### 输出的 JSON 数据格式

```json
{
  "characters": [
    { "id": "宝玉", "name": "贾宝玉", "category": 2 },
    { "id": "黛玉", "name": "林黛玉", "category": 0 }
  ],
  "relationships": [
    { "source": "宝玉", "target": "黛玉", "type": "恋人/知己" }
  ]
}
```
